# Contribuir

Gracias por querer contribuir. Para mantener este proyecto claro y amigable, sigue estas pautas mínimas:

- Abre un _issue_ para discutir cambios grandes antes de trabajar en ellos.
- Haz un fork y crea una rama con un nombre descriptivo: `feature/mi-cambio` o `fix/descripcion`.
- Ejecuta `npm install` y `npm run build` localmente antes de crear PRs.
- Mantén los cambios pequeños y enfocados (un PR = una idea).
- Usa mensajes de commit claros y en inglés o español breve.
- Añade pruebas o pasos para reproducir el problema si aplica.
- Respeta el `CODE_OF_CONDUCT.md` si existe.

Proceso de Pull Request

1. Crea un fork y una rama.
2. Haz tus cambios y asegúrate que `npm run build` pasa.
3. Abre un PR hacia `main` con descripción clara.
4. Responde a comentarios y actualiza el PR si te piden cambios.

Gracias 💜 — cualquier contribución es bienvenida.
